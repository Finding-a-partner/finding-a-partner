# Социальная сеть для поиска напарника
Этот проект представляет собой социальную сеть для поиска напарников. Пользователь может добавляться в друзья, отмечать мероприятия которые хочет посетить и подписываться на группы по интересам. Пользователь или группа может создавать мероприятие. В системе реализована персональная новостная лента в которой появляются мероприятия, созданные друзьями пользователя или группами, на которые он подписан. Также есть чаты для общения с друзьями.
# Архитектура
Это микросервисная архитектура в которой есть: 
1) Auth-service: регистрация и аутентификация пользователей
2) User-service: управление профилями пользователей, добавление и удаление друзей
3) Event-service: создание и управление мероприятиями
4) Group-service: создание и управление группами
5) Message-service: чаты для общения между пользователями
6) Feed-service: формирование персональной новостной ленты
7) API Gateway: маршрутизация запросов, аутентификация с использованием JWT
8) Service Discovery: обнаружение сервисов, реализация через Spring Cloud Eureka
Для запуска и оркестрации сервисов используется Docker. Для мониторинга и сбора метрик подключены Prometheus и Grafana.
# Что было использовано
Spring Boot для разработка микросервисов.
PostgreSQL для хранения данных. 
Docker - контейнеризация.
React TypeScript для создания пользовательского интерфейса.
Prometheus, Grafana для мониторинга и визуализации метрик.
